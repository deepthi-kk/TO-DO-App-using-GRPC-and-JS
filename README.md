# TO-DO application using GRPC and Node.js

1. Used grpc library for Node.JS to create To-Do application
2. The client can add a To-Do item and read all the To-Do items that are added from the memory
3. Used Server-streaming method in grpc to stream all the To-Do items to client when the client makes a call

## To run the code

1. run 
```
npm install @grpc/grpc-js @grpc/proto-loader 
```
